# Zo Config Templates

This folder contains copy/paste templates for Zo Personas and Rules so teams can share the same operating behavior.

## Files

- `personas/tebra-soap-ops.persona.md` — Persona template for practice-manager-facing Tebra SOAP work.
- `rules/tebra-soap-default.rule.md` — Rule template that routes Tebra/Kareo/SOAP requests into the persona behavior.

## How to import into Zo

1. Open [Personas](/?t=settings&s=ai&d=personas).
2. Create a new persona named `Tebra SOAP Ops`.
3. Paste the content from `personas/tebra-soap-ops.persona.md` into the prompt/instructions field.
4. Open [Rules](/?t=settings&s=ai&d=rules).
5. Create a new rule and paste content from `rules/tebra-soap-default.rule.md`.

## Notes

- These are templates, not auto-import files.
- Keep this folder versioned so persona/rule behavior stays consistent across collaborators.
